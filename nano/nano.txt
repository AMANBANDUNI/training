Nano Editor Commands:

1) Undo => Alt + U

2) Redo => Alt + E

3) Paste => ctrl + U

4) Copy => Alt + ^/6

5) Cut => Ctrl + k

6) Rename => ctrl + (\/|)

7) Search(where is) => ctrl + W

8) Search (where next) => Alt + W

9) ctrl + O => To save a file

10) ctrl + X => To exit editor.

11) Mark text => Alt + A

12) Mouse support enable/disable => Alt + M

13) Go to line[Line no., collumn] => ctrl + _

14) Inserting one file into another => ctrl + R
	Type file_name[If file exist in the same Directory you are working on]
		or 
		Complete Location.

15) Alt + (/) => TO jump to the Bottom of the text.

16) alt + \ =>  move curson to top
