Nano Editor Commands:

1) Alt + U		 =>				 Undo

2) Alt + E		 =>				 Redo

3) ctrl + U		 =>				 Paste

4) Alt + ^/6		 =>				 Copy

5) Ctrl + k		 =>				 Cut

6) ctrl + (\/|)		 =>				 Rename

7) ctrl + W		 =>				 Search(where is)

8) Alt + W		 =>				 Search (where next)

9) ctrl + O		 =>				 To save a file

10) ctrl + X		 =>				 To exit editor.

11)  Alt + A		 =>				 Mark text

12)  Alt + M		 =>				 Mouse support enable/disable

13)  ctrl + _		 =>				 Go to line[Line no., collumn]

14)  ctrl + R		 =>				 Inserting one file into another
							 Type file_name[If file exist in the same Directory you are working on]
							 or 
							 Complete Location.

15) Alt + (/)		 =>				 TO jump to the Bottom of the text.

16) alt + \		 =>				 move cursor to top

17) ctrl + c		 =>				 cancel opreration.
