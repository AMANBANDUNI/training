Linux shell/Terminal Basic Commands:

1) pwd => Print working Directory.

(To print the Location you are currently working in )

Example:
 $ pwd
/home/anoj/practice/training/linux


2) cd => change Directory 

(To change/shift to a diffrent Directory/path)

Example: 
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ cd
anoj@anoj-hp-probok-6460b:~$ ls
 ali_git   Documents   examples.desktop   nanobox-codeigniter  'New Folder'   practice   README.md   Templates
 Desktop   Downloads   Music              nanobox-rails         Pictures      Public     snap        Videos
anoj@anoj-hp-probok-6460b:~$ cd practice/training/linux/
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ ls
anyfile.txt  linux.txt


3) cd - => (To move to the last location/path)

4) cd .. => (To move to the parent Directory of the current Dorectory)


5) ls => List

( To get a list of all Folders and files of current Directory.)

Example:
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ ls
anyfile.txt  linux.txt

anoj@anoj-hp-probok-6460b:~$ ls
 ali_git   Documents   examples.desktop   nanobox-codeigniter  'New Folder'   practice   README.md   Templates
 Desktop   Downloads   Music              nanobox-rails         Pictures      Public     snap        Videos



6) ls -a => List -all

( To get a list of all the Directories folders , files including the hidden Directories.)

Example:
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ ls -a
.  ..  anyfile.txt  linux.txt



7) mkdir <directory/folder name>  <directory/folder name> <etc.> => makedirectory

(To make a new  Directory/Directories)

Example: 
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ mkdir myfirstDirectory
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ ls
anyfile.txt  linux.txt  myfirstDirectory


8) rmdir <directory/folder name>  <directory/folder name> <etc.> => remove Directory

(To Remove an empty Directory/Directories)
Example:
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ ls
anyfile.txt  linux.txt  myfirstDirectory
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ rmdir myfirstDirectory/
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ ls
anyfile.txt  linux.txt



9) touch => (To create an empty file)

Example:
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ touch anyfile.txt
anoj@anoj-hp-probok-6460b:~/practice/training/linux$ ls
anyfile.txt  linux.txt


10) copy => ctrl + shift + C

11) paste => ctrl + shift + V



12) rm => remove
(To remove file/files)


13) rm -r => remove recurssively
(To remove a folder which containing files)

14) rm -rf => remove forcefully
(To Remove any file/files/Directory/Directories Forcefully)


15) cp => copy
syntax=> cp <filename> location/path/paths
(To copy files to different path/paths)


16) cp -r => copy recurssively
syntax=> cp -r <Directoryname> location/path/paths
(To copy a complete Directory to different path/paths)


17) clear => (To clear screen)
